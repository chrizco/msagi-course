# MSAGI Course Projects

This repository contains practical Python and AI engineering assignments from the MSAGI online course.

## Structure

- assignments/01_rag_pipeline - RAG pipeline using PDF chunking, OpenAI embeddings, FAISS, and LangChain
- shared/ - reusable course-level helpers or templates if needed later

## Notes

- The local .venv is not committed.
- API keys are stored only in local .env files and are not committed.
- Generated vectorstores are not committed.
- Assignment folders contain notebooks, reports, selected datasets, and outputs.
